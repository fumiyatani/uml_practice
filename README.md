## モデリングとは？
以下を理解した上で進めること
https://www.ogis-ri.co.jp/otc/hiroba/others/ModelingCafe/01.html

対象を模式化 (=図に表す) すること
モデリングの対象を細かく表現せず、直腸的な部分を利用し、重要ではない部分は捨てる。

## モデリングの進め方
1. モデリング対象の全体像を把握する
    - 色々な関わりを考える

2. モデルのコンセプトを決める
    - モデリング対象として欠かせないものがコンセプトになる。そして少ない情報でモデリング対象の重要な特徴を表現する。

3. モデルを作成する
    - 前段で決めたコンセプトにしたがってモデルを作成する
